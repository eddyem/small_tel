\section{Общие работы}
\subsection{Ремонт и строительство}
\begin{itemize}
\item Завершение строительных работ: обшивка стыков сендвич-панелей, подшивка крыши, засыпание траншей и
выравнивание грунта, уборка в помещениях.
\item Организация навеса перед входом в основное помещение.
\item Укрепление настилов из просечки (иметь хотя бы 10-кратный запас прочности).
\item Покраска металлоконструкций (лестницы, настилы, столбы).
\item Восстановление молниеотвода.
\item Замена верхней (южной) секции забора на сплошной усиленный. Ремонт и укрепление остальных секций забора.
\end{itemize}

\subsection{Электрические и монтажные работы}
\begin{itemize}
\item Реконструкция сигнализации, реконструкция камер видеонаблюдения и организация доступа к ним с компьютеров
внутренней сети САО (временно реализовано как сервис ZoneMinder \url{roboserv.sao.ru/zm}, требуется перенести на
выделенный сервер видеорегистрации, заменив на нем windos на Linux).
\item Изготовление и монтаж управляемого освещения подкупольного и ламп плоского поля во второй и третьей башнях.
\item Изготовление и монтаж аварийного отключения питания куполов (во всех трех башнях).
\item Изготовление собственной системы управления куполами <<Astrosib>> с возможностью сброса ошибок.
\item Монтаж концевых выключателей, однозначно обозначающих полное закрытие куполов.
\item Реконструкция системы внутреннего терморегулирования в основном помещении.
\item Установка контакторов (магнитных пускателей) с дистанционным управлением на все важные узлы комплекса для
возможности аварийного удаленного отключения.
\end{itemize}



\section{Работы по вводу в строй телескопа \No\,3}
Проведение электричества и ethernet в подкупольное:
\begin{itemize}
\item  прокладка кабель-каналов;
\item монтаж дополнительных дифференциальных автоматических выключателей и автомата для освещения;
\item монтаж розеток, выключателя и прокладка кабеля.
\end{itemize}

Запуск ИБП и монтаж вычислительного оборудования:
\begin{itemize}
\item проведение отдельной линии, защищенной ИБП, к оборудованию телескопа;
\item монтаж аккумуляторов и запуск ИБП;
\item изготовление управляемого блока розеток;
\item монтаж компьютера и управляемого блока розеток на колонне телескопа;
\item прокладка кабелей питания и коммуникации к узлам телескопа и фотометру.
\end{itemize}

Установка осушителя воздуха:
\begin{itemize}
\item монтаж осушителя на виброгасящей платформе;
\item прокладка вентиляционных труб;
\item подключение удаленного управления и концевика, отключающего осушитель при открытом куполе.
\end{itemize}

\section{Закупки}
(список неполон, т.к. необходимо рассчитать количество металлоконструкций и листов поликарбоната для установки
навеса перед входом в основное здание; количество металлоконструкций и краски для реставрации лестниц и настилов;
комплектующие для заблаговременной закладки силовых и коммуникационных линий к местам установки четвертого
телескопа, а также <<детских>> телескопов)
\subsection{Комплектующие}
Компьютерные (восполнять по мере выхода из строя):
\begin{itemize}
\item стриммер (ленточный накопитель) для архивации данных с сетевого накопителя на жестких дисках;
\item магнитные ленты для стриммера (из расчета 2\,ТБ в год на каждый телескоп);
\item жесткие диски (в среднем~--- один диск в год);
\item аккумуляторы для ИБП (полная замена всех аккумуляторов каждые 3~года);
\item одноплатные ПК (в среднем~--- один в три года);
\item блоки питания (в среднем~--- один в пять лет).
\end{itemize}

Для разработки системы управления куполами <<Astrosib>>:
\begin{itemize}
\item частотный преобразователь VLT Micro Drive FC 51 0.75кВт;
\item трехфазный асинхронный двигатель с тормозом 0.75кВт;
\item абсолютный многооборотный энкодер (10~бит на оборот, 12~битный счетчик оборотов).
\end{itemize}

\subsection{Электрика, подсветка <<плоского поля>>}
\begin{itemize}
\item Кабель силовой $3\times2.5\,$мм${}^2$ (многопроводная жила)~--- 200м.
\item Блоки силовых розеток в кабель-канал (3~блока по 4~розетки).
\item Блоки ethernet-розеток в кабель-канал (2~блока по 2~розетки).
\item Кабель-канал <<Легранд>> для установки розеток (10м).
\item Фурнитура для кабель-канала (повороты, заглушки и т.п.).
\item Дифференциальные автоматы 230\,В, 25\,А, 30\,мА~--- 4~штуки.
\item Трехфазный контактор (магнитный пускатель) 410\,В, напряжение управления 230\,В~--- 4~штуки.
\item Матовый белый полистирол толщиной 4\,мм, $2000\times3000\,$мм~--- 2~листа.
\item Пенопласт листовой $1000\times1000\times50\,$мм, 10~листов.
\item Труба стальная $20\times20\times2\,$мм~--- 15~метров.
\item Краска черная в баллончиках~--- 10\,штук.
\item USB-реле (блок на 4~реле)~--- 5\,штук.
\item Кабель ПВС $2\times0.75\,$мм${}^2$~--- 25\,метров.
\item Кабель-канал $20\times30\,$мм~--- 20\,метров.
\item Патрон для лампы на кабель~--- 3\,штуки.
\end{itemize}
