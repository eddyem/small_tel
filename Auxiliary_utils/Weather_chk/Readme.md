Check meteostation parameters
===========================

    Usage: chkweather [args]
    
            Where args are:
    
      -d, --devname=arg   serial device name
      -h, --help          show this help
      -s, --speed=arg     baudrate (default: 9600)


Output: 
Rain=0/1
Clouds=0/1

Return value:
0 if no rain, 1 if there's rainy
